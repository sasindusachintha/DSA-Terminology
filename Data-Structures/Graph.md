# Graph

## Defintion
A graph is a non-linear data structure consisting of a set of vertices and a set of edges that connect pairs of vertices.   G = (V, E)

---

## key Characteristics
- Non- linear structure
- Can be directed or undirected
- May contain cycles
- Can be connected or disconnected

---

## Types of Graphs
- Undirected Graphs
- Directed Graphs
- weighted Graph

---

## Basic Terminology
- Vertex (Node)
- Edge
- Degree
- Path
- Cycle
- Connected Graph
- Disconnected Graph
- Weight

---

## Graph Representation
- Adjacency Matrix
- Adjacency List

---

## Graph Traversal
- Breadth First Search (BFS)
- Depth First Search (DFS)
